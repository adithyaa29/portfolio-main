Health Guard is a comprehensive web application designed to assess and predict the risk of multiple diseases, focusing on conditions such as diabetes and cardiovascular diseases (CVD). The project employs a user-friendly front end developed using HTML and CSS to ensure a seamless and intuitive user experience. The backend of the application is powered by Python, utilizing advanced algorithms and data analysis techniques to provide accurate risk predictions based on user input and relevant health data.
